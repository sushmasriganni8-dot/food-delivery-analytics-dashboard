<h1 align="center">FoodPulse AI - Food Delivery Analytics Dashboard</h1>

<p align="center">
Interactive Business Intelligence Dashboard developed using Python and Power BI to analyze food delivery sales, revenue trends, customer behavior, and business performance.
</p>

<br>

## Project Overview

This project focuses on transforming raw food delivery data into meaningful business insights using data analytics and visualization techniques.

The dashboard helps in:
- Monitoring sales performance
- Identifying top-selling items
- Understanding transaction behavior
- Tracking revenue trends
- Analyzing peak sales timings
